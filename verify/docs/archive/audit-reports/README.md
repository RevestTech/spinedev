# Archived HTML audit exports

Legacy static HTML snapshots (e.g. FCNow-era reports) kept for history.

**Current practice:** triage and handoff for repositories **scanned by** Tron belong in **those application repos** (managed handoff files, `tron.md`, etc.) — see **`tron/agent_handoff_templates/README.md`** and **`.cursor/rules/tron-scanned-app-handoff.mdc`**.

Do not treat files in this folder as the live assurance record for external products.
