Machines Every host that has executed a Spine agent. Status pill reflects whether the host has reported in via `team.sh up` recently. 'Unregistered' means we've seen cost rows from this host but no instance heartbeat — restart `team.sh down && team.sh up` to register.

Engagements An engagement is a piece of client work tracked end-to-end: requirements -> hardening -> planning -> approval -> execution -> delivery. Click 'New Engagement' to submit a brief and let the product role take it from there.

Fleet Every Spine instance that has reported in. Alive = heartbeat under 60s old. Stale = 60-180s. Lost = over 180s without a heartbeat. Stopped = explicitly shut down with team.sh down.

Recent invocations The last 50 times an agent picked up a directive and tried to do something with it. Green = succeeded, red = killed by daemon (timeout/stall), amber = still in flight.

Cost by role Which agent roles burned the most money. Useful for spotting an expensive role that should be moved to a cheaper tier.

Cost by model Which AI models you actually called and what they cost per million tokens. Empty when the executor doesn't emit usage data (today only Claude Code does).

Active workers All worker handles the watcher has seen at least once. Lifetime invocations and lifetime cost are cumulative across that worker's history.

Event stream Every lifecycle event in time order. Click a row's payload cell to expand the full JSON. The donut shows the distribution of event types.
